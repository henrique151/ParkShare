# ParkShare 2

Cadastro de Usuários: Permite que usuários (moradores e externos) se cadastrem na plataforma com informações pessoais, de contato e do veículo.
Disponibilização de Vagas: Proprietários de vagas podem registrar seus horários de disponibilidade e as condições de uso.
Gestão do Condomínio: Síndicos e administradores podem cadastrar o condomínio, definir regras, aprovar solicitações e fiscalizar o uso das vagas.
Busca e Reserva de Vagas: Usuários externos podem pesquisar por vagas disponíveis, solicitar reservas e confirmar o uso.
Interação Direta: A plataforma permite a negociação e reserva de vagas diretamente entre os usuários, com o síndico atuando como mediador.
Sistema de Reservas: O sistema gerencia as reservas, registrando informações como data, horário e status (pendente, aprovada, encerrada).
Controle de Veículos: Apenas veículos cadastrados podem utilizar as vagas, garantindo a segurança.
Geolocalização: O sistema utiliza dados de localização para facilitar a busca por vagas em diferentes regiões.
